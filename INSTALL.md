# Installation

Assuming you have installed [^1] Idris [^2] correctly:


## Fetch dependencies

```sh
bash fetch-deps.sh
```

## Build library

```sh
make build
```

## Install the package into Idris' package eco-system.

```sh
make install
```

[^1]: https://github.com/idris-lang/Idris-dev/wiki/Installation-Instructions

[^2]: https://www.idris-lang.org
